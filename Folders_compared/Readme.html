Folder Comparison Tool - User Guide / Guía de Usuario

🇬🇧 English

Introduction

This tool allows you to compare the contents of two folders and identify which files are missing in each. Optionally, you can filter the comparison by file extension. The results are saved both as a TXT file and an Excel spreadsheet for easy analysis.

1. Installing Python

  1. Visit python.org.
  2. Download the Windows installer.
  3. IMPORTANT: Make sure to check "Add Python to PATH" before installing.

2. Installing Required Libraries

  1. Open PowerShell.
  2. Type the following command and press Enter:
    pip install openpyxl
  3. This installs the library needed to create Excel files.

3. How to Use the Script

  1. Run the script: Folders_Compared.py.
  2. Select Folder A and Folder B using the interface.
  3. Optionally enter a file extension (e.g., .txt) to limit the comparison.
  4. Click Compare. The results will show which files are missing from each folder.
  5. The comparison is saved to a resultado_comparacion.txt file and an comparacion_folders.xlsx file in the script directory.

Introducción

Esta herramienta te permite comparar el contenido de dos carpetas y ver qué archivos faltan en cada una. Puedes filtrar por extensión de archivo si lo deseas. El resultado se guarda como archivo TXT y en una hoja de Excel para facilitar el análisis.

1. Instalar Python

  1. Ve a python.org.
  2. Descarga el instalador para Windows.
  3. IMPORTANTE: Asegúrate de marcar "Add Python to PATH" antes de instalar.

2. Instalar Librerías Necesarias

  1. Abre PowerShell.
  2. Escribe y ejecuta:
    pip install openpyxl
  3. Esto instalará la librería necesaria para generar archivos de Excel.

3. Cómo Usar el Script

  1. Ejecuta el script: Folders_Compared.py.
  2. Selecciona la Carpeta A y la Carpeta B desde la interfaz.
  3. Opcionalmente, introduce una extensión (por ejemplo: .txt).
  4. Haz clic en Comparar. Verás qué archivos faltan en cada carpeta.
  5. El resultado se guarda como resultado_comparacion.txt y comparacion_folders.xlsx en el directorio del script.
