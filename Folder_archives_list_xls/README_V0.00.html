Folder File Listing Tool - User Guide / Guía de Usuario

🇬🇧 English

1. Introduction

This tool allows you to select a folder and automatically generate an Excel file listing all files in that folder (including subfolders), along with their full path and size in kilobytes.

2. Functional Description

  • Opens a dialog to select the folder to analyze.
  • Walks through the folder and all subfolders using os.walk().
  • For each file, stores the file name, full path, and size in KB.
  • Generates a Pandas DataFrame and exports it to listado_archivos.xlsx in the same folder.
  • Displays a confirmation message upon successful export.

3. Installing Python

  1. Go to python.org.
  2. Download and install Python. Make sure to check "Add Python to PATH".

4. Installing Required Libraries

  1. Open PowerShell.
  2. Run:
    pip install pandas openpyxl

5. How to Use

  1. Run Listado_Archivos_Carpeta.py.
  2. Select the folder you want to list.
  3. The script will create an Excel file named listado_archivos.xlsx in that folder.
  4. A confirmation message will appear once the file is saved.

🇪🇸 Español

1. Introducción

Esta herramienta te permite seleccionar una carpeta y generar automáticamente un archivo Excel con el listado de todos los archivos dentro de ella (incluyendo subcarpetas), con su ruta completa y tamaño en kilobytes.

2. Descripción Funcional

  • Abre un cuadro de diálogo para seleccionar la carpeta a analizar.
  • Recorre la carpeta y sus subcarpetas usando os.walk().
  • Para cada archivo, guarda su nombre, ruta completa y tamaño en KB.
  • Genera un DataFrame de Pandas y lo exporta como listado_archivos.xlsx en la misma carpeta.
  • Muestra un mensaje de confirmación al finalizar.

3. Instalar Python

  1. Visita python.org.
  2. Descarga e instala Python. Asegúrate de marcar "Add Python to PATH".

4. Instalar Librerías Necesarias

  1. Abre PowerShell.
  2. Ejecuta:
    pip install pandas openpyxl

5. Cómo Usar

  1. Ejecuta Listado_Archivos_Carpeta.py.
  2. Selecciona la carpeta que deseas listar.
  3. El script creará un archivo Excel llamado listado_archivos.xlsx en esa misma carpeta.
  4. Aparecerá un mensaje de confirmación al guardar el archivo.
