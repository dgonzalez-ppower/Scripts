1. Installing Python

  1. Download Python from python.org.
  2. Run the installer and make sure to check "Add Python to PATH".
  3. Complete the installation.

2. Installing Required Libraries

  1. Open PowerShell (Press Start, type PowerShell).
  2. Run this command:
    pip install pytesseract pdf2image pymupdf
  3. Install Tesseract OCR from GitHub.

3. How to Use the Script

  1. Run the script: PDF_ocr_testv0.11.py.
  2. Select a PDF file when asked.
  3. The script will analyze the first page, then scan and mark each page.
  4. Red = known match, Green = inferred match.
  5. The output will be saved as yourfile_marked.pdf.
