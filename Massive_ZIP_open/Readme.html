ZIP Extractor - User Guide / Guía de Usuario

🇬🇧 English

1. Installing Python

  1. Go to python.org.
  2. Download the installer for Windows.
  3. IMPORTANT: Check "Add Python to PATH" before clicking Install.

2. Installing Required Libraries

  1. Open PowerShell (Press Start, type PowerShell).
  2. Run the following command:
    pip install tk
  3. That's all you need for this tool!

3. How to Use the Script

  1. Run the script: Massive_ZIP_open.py (double-click or use Python).
  2. A window will appear. Click Select Folder.
  3. Select a folder that contains one or more .zip files.
  4. Each ZIP file will be extracted into its own subfolder.
  5. A message will confirm how many ZIPs were extracted.

🇪🇸 Español

1. Instalar Python

  1. Ve a python.org.
  2. Descarga el instalador para Windows.
  3. IMPORTANTE: Marca la opción "Add Python to PATH" antes de instalar.

2. Instalar Librerías Necesarias

  1. Abre PowerShell (Inicio > escribe PowerShell).
  2. Ejecuta el siguiente comando:
    pip install tk
  3. ¡Eso es todo lo que necesitas para esta herramienta!

3. Cómo Usar el Script

  1. Ejecuta el script: Massive_ZIP_open.py (doble clic o con Python).
  2. Aparecerá una ventana. Haz clic en Seleccionar carpeta.
  3. Selecciona una carpeta que contenga archivos .zip.
  4. Cada ZIP se extraerá en su propia subcarpeta con el mismo nombre.
  5. Un mensaje confirmará cuántos ZIPs fueron extraídos.
