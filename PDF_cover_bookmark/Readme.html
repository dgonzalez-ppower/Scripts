Introduction

This script adds a bookmark titled Cover to the first page of every PDF file in a selected folder. It saves the modified PDFs into another folder, making it easy to navigate large PDF collections where a title or intro page needs highlighting.

1. Installing Python

  1. Download Python from python.org.
  2. Run the installer and make sure to check "Add Python to PATH".

2. Installing Required Libraries

  1. Open PowerShell.
  2. Run the following command:
    pip install pypdf

3. How to Use the Script

  1. Run the script: Marcador_cover_PDF.py.
  2. When prompted, select the folder with your original PDFs.
  3. Then select the folder where the modified PDFs should be saved.
  4. The script will add a bookmark to the first page of each file and save the result.
  5. A confirmation message will appear once all PDFs have been processed.
