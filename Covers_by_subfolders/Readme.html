Introduction

This tool creates personalized cover pages for each folder or subfolder using a Word document template. It replaces the placeholder {{FOLDER_NAME}} in the template with the folder name and exports the result as a Word and PDF file. You can choose between generating covers just for first-level folders or all nested subfolders recursively.

1. Installing Python

  1. Go to python.org.
  2. Download the Windows installer.
  3. IMPORTANT: Check the option "Add Python to PATH" before clicking install.

2. Installing Required Libraries

  1. Open PowerShell.
  2. Type and run:
    pip install python-docx docx2pdf
  3. Make sure Microsoft Word is installed (required for PDF conversion).

3. How to Use the Script

  1. Run the script: caratulas_subcarpetas_v0.02.py.
  2. Select your Word template (.docx) that includes the text {{FOLDER_NAME}}.
  3. Select the root folder that contains subfolders.
  4. Choose whether to generate covers only for first-level subfolders or recursively for all.
  5. Click Generate Covers. A Word and PDF cover will be created in each folder.
