Introduction

This script reprints and flattens every PDF in a selected folder. It uses Ghostscript to regenerate the PDF and PDFtk to flatten the content (i.e., make annotations, forms, and layers non-editable). It's ideal for preparing PDFs for printing or distribution.

1. Installing Python

  1. Download Python from python.org.
  2. Run the installer and check "Add Python to PATH" before installation.

2. Installing External Tools

  1. Install Ghostscript (ensure gswin64c or gswin32c is available in PATH).
  2. Install PDFtk and ensure pdftk works from the command line.

3. How to Use the Script

  1. Run PDF_reimp_flattenv0.08.py.
  2. Select the folder containing your PDF files when prompted.
  3. The script will create a new folder named like reimp_YYYYMMDD_HHMMSS.
  4. Each original PDF will be:
    • Reprinted â†’ saved as originalname_reimp.pdf
    • Flattened â†’ saved as originalname_reimp_aplanado.pdf
  5. Check the output folder for results and logs in the terminal.
