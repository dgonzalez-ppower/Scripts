PDF Bookmark Splitter - User Guide / Guía de Usuario

🇬🇧 English

Introduction

This script allows you to take a PDF file with bookmarks and automatically split it into separate PDF files. Each new file is saved in a folder named after its corresponding bookmark. This is useful for batch processing documents like reports, certificates, or invoices where each section needs to be isolated.

1. Install Python

  1. Go to python.org/downloads.
  2. Download the version for Windows.
  3. Run the installer. Important: Check "Add Python to PATH" before installing.

2. Install Required Libraries

  1. Open PowerShell (press Start, type "PowerShell").
  2. Type this command and press Enter:
    pip install pymupdf PyPDF2
  3. Wait for the installation to finish.

3. How to Use

  1. Download and save the script (e.g., Bookmark_2_Folders_2_Files_v0.08.py).
  2. Double-click it or run from PowerShell:
    python Bookmark_2_Folders_2_Files_v0.08.py
  3. Click the Select PDF File button and choose a PDF with bookmarks.
  4. The tool will create folders for each bookmark and extract the relevant PDF pages into them.

🇪🇸 Español

Introducción

Este script permite tomar un archivo PDF con marcadores y dividirlo automáticamente en archivos PDF separados. Cada nuevo archivo se guarda en una carpeta con el nombre del marcador correspondiente. Es ideal para procesar documentos por lotes como informes, certificados o facturas que necesitan dividirse por secciones.

1. Instalar Python

  1. Ve a python.org/downloads.
  2. Descarga la versión para Windows.
  3. Ejecuta el instalador. Importante: Marca "Add Python to PATH" antes de instalar.

2. Instalar Librerías Necesarias

  1. Abre PowerShell (haz clic en Inicio y escribe "PowerShell").
  2. Escribe el siguiente comando y presiona Enter:
    pip install pymupdf PyPDF2
  3. Espera a que termine la instalación.

3. Cómo Usar

  1. Descarga y guarda el script (por ejemplo: Bookmark_2_Folders_2_Files_v0.08.py).
  2. Haz doble clic o ejecútalo desde PowerShell:
    python Bookmark_2_Folders_2_Files_v0.08.py
  3. Haz clic en el botón Select PDF File y elige un PDF con marcadores.
  4. La herramienta creará carpetas por cada marcador y extraerá las páginas correspondientes.
