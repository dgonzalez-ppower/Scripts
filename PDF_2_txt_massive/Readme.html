PDF to TXT Converter - User Guide / Guía de Usuario

🇬🇧 English

Introduction

This tool allows you to convert multiple PDF files into plain text (.txt) files using a simple graphical interface. You select a folder with PDFs and a destination folder, and the tool extracts text from each PDF and saves it as a .txt file.

1. Installing Python

  1. Go to python.org.
  2. Download the Windows installer.
  3. IMPORTANT: Make sure to check "Add Python to PATH" before installing.

2. Installing Required Libraries

  1. Open PowerShell.
  2. Run the command:
    pip install PyPDF2

3. How to Use the Script

  1. Run the script: pdf_to_txt_converter_gui.py.
  2. Use the GUI to select your source folder with PDF files.
  3. Select a destination folder where the .txt files should be saved.
  4. Click Convert PDFs to TXT.
  5. The tool will extract text from each PDF and save it in the selected destination folder.

Introducción

1. Instalar Python

  1. Visita python.org.
  2. Descarga el instalador para Windows.
  3. IMPORTANTE: Asegúrate de marcar "Add Python to PATH" antes de instalar.

2. Instalar Librerías Necesarias

  1. Abre PowerShell.
  2. Ejecuta el siguiente comando:
    pip install PyPDF2

3. Cómo Usar el Script

  1. Ejecuta el script: pdf_to_txt_converter_gui.py.
  2. Usa la interfaz para seleccionar la carpeta de origen con archivos PDF.
  3. Selecciona la carpeta de destino donde se guardarán los archivos .txt.
  4. Haz clic en Convert PDFs to TXT.
  5. La herramienta extraerá el texto de cada PDF y lo guardará como archivo .txt en la carpeta seleccionada.
