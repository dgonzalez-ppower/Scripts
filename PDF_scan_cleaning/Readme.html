Introduction

This script processes one or multiple PDF files, improving the quality of scanned pages and applying OCR (text recognition). It enhances contrast and brightness, recreates the PDF from improved images, and applies OCR using OCRmyPDF, generating a clean, searchable PDF.

2. Detailed Internal Functional Description

The script works in the following way:

  • It prompts the user to select a single PDF file or a folder containing multiple PDFs.
  • If a folder is selected, the user can choose whether to include subfolders.
  • It processes each PDF by:
    • Extracting each page as an image.
    • Applying brightness, contrast, and enhancement filters to clean and improve scanned quality.
    • Recreating a new PDF from the enhanced images.
    • Running OCR on the cleaned PDF to generate a text-searchable version.
  • Outputs two versions for each original PDF:
    • One cleaned but without OCR (filename_FINAL.pdf).
    • One cleaned and searchable with OCR (filename_FINAL_OCR.pdf).
  • Displays progress messages during processing and handles errors gracefully.

3. Installing Python

  1. Download Python from python.org.
  2. Run the installer and ensure you check "Add Python to PATH".

4. Installing Required Libraries and Tools

  1. Open PowerShell.
  2. Run:
    pip install pymupdf Pillow ocrmypdf
  3. Ensure OCRmyPDF is installed and functional (it may require additional setup, such as installing Ghostscript, Tesseract, etc.).

5. How to Use the Script

  1. Run the script: limpiar_escaneo_batch_v0.11.py.
  2. Select a PDF file or a folder containing multiple PDFs.
  3. If a folder is selected, choose whether to include subfolders.
  4. Select a destination folder for the output PDFs.
  5. The script will enhance page images, recreate the PDF, apply OCR, and save results into the destination folder.
  6. Each processed PDF will have two versions:
    • filename_FINAL.pdf: cleaned but without OCR.
    • filename_FINAL_OCR.pdf: cleaned and searchable with OCR.
