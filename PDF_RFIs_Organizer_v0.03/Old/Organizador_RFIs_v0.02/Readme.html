📂 README - Organizador de RFIs

📘 Versión en Español

Este proyecto es una herramienta interactiva con interfaz gráfica (GUI) desarrollada en Python para facilitar el procesamiento, organización y clasificación de archivos PDF de RFIs (Request For Information), utilizando un archivo Excel como índice de referencia.

⚙️ Descripción Funcional

El sistema está compuesto por dos scripts principales:

  • Organizador_RFIs.py: GUI principal para seleccionar carpeta de PDFs e índice Excel. Ejecuta el proceso, muestra logs y genera un reporte final.
  • Organizador_RFIs_funciones.py: Contiene funciones para extraer bookmarks, dividir PDFs, moverlos y generar logs.

📦 Requisitos

  • Python 3.7+
  • Librerías: pandas, PyPDF2

🚀 Cómo usar

  1. Ejecutar Organizador_RFIs.py
  2. Seleccionar carpeta con PDFs
  3. Seleccionar archivo Excel
  4. Presionar "Procesar"
  5. Revisar resultado y logs generados

📁 Estructura del Excel

  • Nombre RFI
  • Subindice
  • Nombre PDF Destino
  • Enlace Destino

📘 English Version

This project is a Python-based GUI tool for organizing and processing RFI (Request For Information) PDF files, using a reference Excel file. It supports auto-splitting PDFs using bookmarks, renaming and moving them to target folders.

⚙️ Functional Description

The project consists of two main scripts:

  • Organizador_RFIs.py: The main GUI script that lets users choose a PDF folder and Excel index, processes files, displays logs, and generates a final report.
  • Organizador_RFIs_funciones.py: Contains core logic for extracting bookmarks, splitting PDFs, moving them and generating logs.

📦 Requirements

  • Python 3.7+
  • Libraries: pandas, PyPDF2

🚀 How to Use

  1. Run Organizador_RFIs.py
  2. Select folder containing PDFs
  3. Select Excel index file
  4. Click "Procesar"
  5. Check generated logs and report

📁 Excel Structure

  • RFI Name
  • Subindex
  • Target PDF Name
  • Target Path
