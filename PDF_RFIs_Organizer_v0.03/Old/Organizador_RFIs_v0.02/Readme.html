English Version

This project is a Python-based GUI tool for organizing and processing RFI (Request For Information) PDF files, using a reference Excel file. It supports auto-splitting PDFs using bookmarks, renaming and moving them to target folders.

锔 Functional Description

The project consists of two main scripts:

  • Organizador_RFIs.py: The main GUI script that lets users choose a PDF folder and Excel index, processes files, displays logs, and generates a final report.
  • Organizador_RFIs_funciones.py: Contains core logic for extracting bookmarks, splitting PDFs, moving them and generating logs.

Requirements

  • Python 3.7+
  • Libraries: pandas, PyPDF2

How to Use

  1. Run Organizador_RFIs.py
  2. Select folder containing PDFs
  3. Select Excel index file
  4. Click "Procesar"
  5. Check generated logs and report

  • RFI Name

  • Target PDF Name
